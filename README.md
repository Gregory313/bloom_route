# BloomRoute: Decentralized Overlay Network

Децентрализованная оверлейная сеть с вероятностной маршрутизацией на основе фильтров Блума.

## Основные возможности
* **Вероятностная маршрутизация:** использование фильтров Блума для оценки «близости» к целевому ID.
* **Обнаружение соседей:** автоматическая поддержка таблицы маршрутизации через служебные пакеты (COMM).
* **Асинхронность:** обработка событий через системный вызов `select`.
* **Интерактивный CLI:** встроенная оболочка для отправки сообщений и мониторинга сети.
* **Транспорт:** работа поверх UDP.


## Возможности
- Автоматическое обнаружение соседей
- Маршрутизация на основе фильтров Блума
- Асинхронная работа (select)
- Интерактивная командная оболочка
- Поддержка начальных контактов
- Локальное тестирование


## Структура проекта
* `src/bloom.c` — Реализация фильтров Блума.
* `src/neighbors.c` — Управление таблицей соседей.
* `src/control.c` — Логика обработки пакетов COMM/DATA.
* `src/utils.c` — Сетевые функции.
* `src/main.c` — Точка входа и CLI.
* `Makefile` — Сценарий сборки.

## Параметры
- `-p` - порт UDP (обязательный)
- `-i` - 32-битный ID узла в hex (обязательный)
- `-c` - контакт в формате ip:порт (можно несколько)

## Команды

send <id> <текст> - отправить сообщение
neighbors - показать соседей
exit - выход

ID можно указывать с 0x или без.

## Пример сессии

Терминал A:
$ ./build/node -p 9000 -i 0x11111111

    neighbors
    No neighbors.

Терминал B:
$ ./build/node -p 9001 -i 0x22222222 -c 127.0.0.1:9000

    neighbors
    Neighbors:
    0x11111111 (127.0.0.1:9000) last updated 1234567890
    send 0x11111111 Hello from B!

Терминал A:
Received DATA for me (0x11111111) from 0x22222222: 'Hello from B!'

## Конфигурация (src/conf.h)

BLOOM_M 1024 - размер фильтра Блума
BLOOM_K 8 - количество хеш-функций
BLOOM_C 3 - начальное значение ячейки
TIMEOUT_NEIGHBOR_SEC 60 - время жизни соседа (сек)
COMM_SEND_INTERVAL_SEC 10 - интервал рассылки COMM (сек)
MAX_CONTACTS 16 - максимум начальных контактов
